About

I am a pipeline engineer and bioinformatician at CRUK National Biomarker Centre in Manchester. My work focuses on epigenomics and DNA methylation analysis in the context of cancer biomarker discovery — designing, building, and maintaining production Nextflow pipelines for whole-genome bisulfite sequencing (WGBS), MBD-seq enrichment assays, T7 demultiplexing, and dilution-based methylation workflows.

Previously at the Prinses Máxima Centrum in Utrecht, I built Snakemake pipelines for metagenomics research into human gut and tumour microbiomes in the context of paediatric brain tumours — combining 16S/WGS sequencing, taxonomic profiling, and clinical sample metadata to support translational research.

I build workflows that are reproducible, containerised, and maintainable across SLURM HPC environments. Alongside pipeline engineering, I develop R packages for methylation data analysis using Bioconductor conventions and maintain CI/CD workflows for automated pipeline testing with GitHub Actions.

WGBS / nf-core methylseq Pipeline

GitHub ↗

End-to-end whole-genome bisulfite sequencing pipeline in Nextflow DSL2, following nf-core methylseq conventions. Covers adapter trimming (Trim Galore), bisulfite alignment (Bismark), deduplication, CpG extraction, and DMR calling. Configurable for SLURM HPC and AWS Batch via Singularity containers. Uses public ENCODE WGBS data for demonstration.

Nextflow nf-core Bismark Singularity

HPC Nextflow Templates

GitHub ↗

Production-ready Nextflow DSL2 skeleton workflows for SLURM HPC environments. Covers executor configuration, per-process resource profiles, Singularity execution, environment module loading, and GitHub Actions CI for automated pipeline validation.
